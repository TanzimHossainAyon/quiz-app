# 🧠 QuizMaster — Interactive Quiz App

A beautifully designed interactive quiz application with animations, timer, hints, scoring system and confetti effects!

## 👤 Author
**Tanzim Hossain Ayon** | CSE Graduate · BRAC University · IEEE Researcher  
🌐 [Portfolio](https://tanzimhossainayon.vercel.app)

## 🚀 Features
- 6 categories — AI/ML, Science, Technology, Mathematics, General, All
- 26 questions with shuffled options
- ⏱️ 15-second countdown timer per question
- 🏆 Score system with speed bonuses
- 💡 Hint system (3 hints per game)
- 🔥 Streak tracking & best streak
- 🎊 Confetti animation on high scores
- 📊 Detailed result screen with stats
- ✨ Particle animation background
- 🖱️ Custom cursor
